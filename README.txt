This is a test frontend server that connects to the test-backend. Purpose: to test change detection model.

This frontend has four endpoints:
- v1/v1/json: version 1 of the frontend querying version 1 of the backend
- v1/v2/json: version 1 of the frontend querying version 2 of the backend
- v2/v1/json: version 2 of the frontend querying version 1 of the backend
- v2/v2/json: version 2 of the frontend querying version 2 of the backend
